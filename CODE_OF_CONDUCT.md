# 贡献者公约

本项目采用 [Contributor Covenant](https://www.contributor-covenant.org/) 2.1 版本
作为行为准则。参与本项目即表示你同意遵守此准则。

## 我们的承诺

为了营造开放和友好的环境，我们作为贡献者和维护者承诺：

- 让每个人都能享受无骚扰的体验，无论年龄、体型、残疾、民族、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、种族、宗教或性取向如何。
- 通过公开接受和展示期望的行为，为社区树立正面榜样。
- 对不当行为采取明确反对的态度。

## 适用标准

有助于创造积极环境的行为包括：

* 使用欢迎和包容的语言
* 尊重不同的观点和经验
* 建设性地接受批评
* 关注对社区最有利的事情
* 对其他社区成员表示同理心

不可接受的行为包括：

* 使用性化语言或图像，以及不受欢迎的性关注或骚扰
* 恶意评论、侮辱/贬损性评论，以及政治攻击
* 公开或私下骚扰
* 未经明确许可发布他人的私人信息（如物理地址或电子邮箱）
* 在专业场合被合理认为不适当的其他行为

## 执行职责

项目维护者有责任和权利删除、编辑或拒绝与本行为准则不符的评论（commits）、代码、wiki 编辑、issue 和 PR，或暂时或永久禁止任何他们认为行为不当、威胁、冒犯或有害的贡献者。

## 归属

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/) 版本 2.1，
可在 https://www.contributor-covenant.org/version/2/1/code_of_conduct.html 查看。

## 联系方式

如有关于本行为准则的问题或需要报告违规行为，请通过以下方式联系项目维护者：

* **Email**: gqy20@example.com
* **GitHub Issues**: [创建 Issue](https://github.com/gqy20/gearbox/issues)

所有报告都将被保密处理并认真调查。
